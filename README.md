#  Description

This is an interactive text-based adventure game where the player makes choices that affect the outcome of the story. The game tracks the player's score, provides information about events after every move, and uses random factors to make each playthrough unique.

# Features
Output text to the console.
Import and use Python modules (time, random).
Use the input function with conditional statements (if, while) to create an interactive program.
Handle invalid input with a conditional loop.
Maintain a variable to create and update game state.
Conditions to terminate the program when certain events occur.
The player receives information about what is happening at the start of the game and after every move.
The player receives information about their score at the end of each turn.
time.sleep is used to create delays between messages so they aren't all printed at once.
The random.choice or random.randint function is used to influence the game, making each playthrough different.
The player's choices affect:
Where the player goes
Who they meet
What they do after meeting another character
Whether to restart or exit after the game ends
Invalid input is not accepted; the player gets multiple chances to enter a valid choice.
Total score variable tracks the player's score, which increases or decreases based on game events.
The game ends when conditions are met, such as score, number of turns, or elapsed time.
The program runs from start to finish without crashing or displaying errors.
# How to Play 
Run the game using Python.
Follow the prompts and make choices by typing numbers.
Your score changes based on your decisions.
The game ends when you reach a conclusion based on your choices.
